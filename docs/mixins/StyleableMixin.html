This mixin adds the capability to use common , nested in the elements markup, like so:

<nn-input-text>
  <style slot="style">
   #native {
     color: blue;
   }
  </style>
</nn-input-text>

That allows developers to pierce the shadow DOM and override all of the elements styles using familiar syntax.
